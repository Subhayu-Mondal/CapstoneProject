# Event Feedback System

## Description
A full-stack web application where users can register for events and submit feedback.

## Tech Stack
- PHP
- MySQL
- XAMPP
- Bootstrap

## Setup Instructions
1. Copy project into htdocs/php_practice/
2. Start Apache and MySQL in XAMPP
3. Open phpMyAdmin
4. Create database: event_system
5. Import sql/capstone_db.sql
6. Open browser:
   http://localhost/php_practice/capstoneproject-day-28/
